Ans: Temperature decreases
$\Delta Q = 0 \Rightarrow \Delta U = -\Delta W$. System expands $\Rightarrow \Delta W > 0$, so $\Delta U < 0$. Since $\Delta U = nC_v\Delta T$, a negative $\Delta U$ means $\Delta T < 0$ (temperature drops).

Path 2 (Isothermal): $\Delta U_2=0$, $Q_2 = W_2 = P_i V_i \ln(P_i/P_f) = 3 \times 2 \ln(3/1) = 6 \ln 3 \text{ atm L}$.
Total $Q = Q_1 + Q_2$.

Section B: Isothermal, Adiabatic, and Polytropic Processes
13.
Ans: $5741 \text{ J}$

14.
Ans: $18.38 \text{ atm}$
$P_1V_1^\gamma = P_2V_2^\gamma \Rightarrow 1 \times (16)^{1.4} = P_2 \times (2)^{1.4} \Rightarrow P_2 = (16/2)^{1.4} = 8^{1.4} = 8^{7/5} = (2^3)^{7/5} = 2^{21/5} \approx 18.38 \text{ atm}$.
15.
Ans: $1.5$
$P \propto T^3 \Rightarrow PT^{-3} = \text{const}$. For adiabatic, $P^{1-\gamma}T^\gamma = \text{const} \Rightarrow PT^{\frac{\gamma}{1-\gamma}} = \text{const}$.
Equating powers: $\frac{\gamma}{1-\gamma} = -3 \Rightarrow \gamma = -3 + 3\gamma \Rightarrow 2\gamma = 3 \Rightarrow \gamma = 1.5$.
16.
Ans: $+9224 \text{ J}$ (on the gas)
Diatomic $\Rightarrow \gamma = 1.4$. $T_2 = T_1(V_1/V_2)^{\gamma-1} = 300 \times (4)^{0.4} \approx 522 \text{ K}$.
$W = \frac{nR(T_1-T_2)}{\gamma - 1} = \frac{2 \times 8.31 \times (300 - 522)}{0.4} \approx -9224 \text{ J}$. (Work done ON gas is positive magnitude).
17.
Ans: $1200 \text{ K}$ or $927^\circ\text{C}$
$T_1 = 300 \text{ K}$. $T_2 = T_1(V_1/V_2)^{\gamma-1} = 300 \times (8)^{\frac{5}{3}-1} = 300 \times 8^{2/3} = 300 \times 4 = 1200 \text{ K}$.
18.
Ans: Proof
Isothermal: $PV = c \Rightarrow P dV + V dP = 0 \Rightarrow (dP/dV)_{\text{iso}} = -P/V$.
Adiabatic: $PV^\gamma = c \Rightarrow \gamma P V^{\gamma-1} dV + V^\gamma dP = 0 \Rightarrow (dP/dV)_{\text{ad}} = -\gamma(P/V) = \gamma (dP/dV)_{\text{iso}}$.
19.
Ans: $0.5 nRT_1$
Polytropic with $x=2$. $W = \frac{P_1V_1 - P_2V_2}{x-1}$. Since $P_1V_1^2 = P_2(2V_1)^2 \Rightarrow P_2 = P_1/4$.
$W = \frac{P_1V_1 - (P_1/4)(2V_1)}{2-1} = P_1V_1 - 0.5P_1V_1 = 0.5P_1V_1 = 0.5 nRT_1$.
20.
Ans: $C = C_v + 2R$
$PT = c \Rightarrow P(PV/nR) = c \Rightarrow P^2V = \text{const} \Rightarrow PV^{1/2} = \text{const}$. Thus, $x = 1/2$.
$C = C_v + \frac{R}{1-x} = C_v + \frac{R}{1-0.5} = C_v + 2R$.

$W = nRT \ln(V_2/V_1) \Rightarrow 2000 = 1 \times 8.31 \times 300 \ln(V_2/V_1) \Rightarrow \ln(V_2/V_1) = 2000 / 2493 \approx 0.802$.
$V_2/V_1 = e^{0.802} \approx 2.23$.

$1 \text{ atm L} \approx 101.3 \text{ J} \Rightarrow W = 20 \times 101.3 = 2026 \text{ J}$.

Ans: $\Delta U = 0, \Delta H = 0, W_{by} \approx 5706 \text{ J}, q \approx 5706 \text{ J}$

$W_{by} = nRT \ln(P_1/P_2) = 1 \times 8.314 \times 298 \times \ln(10/1) \approx 5706 \text{ J}$. By 1st Law, $q = \Delta U + W_{by} = 5706 \text{ J}$.
24.
Ans: $\frac{P_0^2}{4\alpha nR}$
$T = \frac{PV}{nR} = \frac{(P_0 - \alpha V)V}{nR} = \frac{P_0 V - \alpha V^2}{nR}$. For max $T$, $dT/dV = 0 \Rightarrow P_0 - 2\alpha V = 0 \Rightarrow V = P_0 / 2\alpha$.
Substitute $V$ back: $T_{max} = \frac{P_0(P_0/2\alpha) - \alpha(P_0/2\alpha)^2}{nR} = \frac{P_0^2}{4\alpha nR}$.

$W = (P_2-P_1)(V_2-V_1)$. Monoatomic $\Rightarrow C_v = 3R/2, C_p = 5R/2$.
Heat added $Q_{in}$ occurs during isochoric heating $Q_1 = \frac{3}{2}V_1(P_2-P_1)$ and isobaric expansion $Q_2 = \frac{5}{2}P_2(V_2-V_1)$. $\eta = W / (Q_1 + Q_2)$.

57.
Ans: Conceptual
Kelvin-Planck: It's impossible to build an engine operating in a cycle that absorbs heat and converts it ALL into work (needs a sink).
Clausius: Heat cannot flow spontaneously from cold to hot without external work.
Equivalence: A violation of one leads to the violation of the other (e.g., passing 100% work from a KP violator to a refrigerator violates Clausius).
