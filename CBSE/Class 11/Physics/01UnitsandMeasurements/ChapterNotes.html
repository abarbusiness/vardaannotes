01.3 Measurement and Errors

Measurement: Comparison of unknown quantity with a known standard quantity of same kind.

Error: Difference between true value and measured value.

$\text{Error} = \text{Measured Value} - \text{True Value}$

Accuracy and Precision

Accuracy: How close a measured value is to the true value.

Precision: How close repeated measurements are to each other (reproducibility).

Key Distinction: A measurement can be precise but not accurate!

Example: If true value = 10.0 cm

  • Measurements: 10.1, 10.0, 9.9 cm â†’ High accuracy, moderate precision
  • Measurements: 12.1, 12.0, 12.1 cm â†’ Low accuracy, high precision

Types of Errors

1. Systematic Errors

Definition: Errors that occur consistently in the same direction (always positive or always negative).

Causes:

  • Instrumental Error: Faulty calibration, zero error in instruments
  • Observational Error: Improper technique, parallax error
  • Environmental Error: Temperature/pressure variations

Characteristics:

  • Can be reduced by proper calibration
  • Can be detected and corrected
  • Do NOT cancel out on averaging

2. Random Errors

Definition: Errors that vary randomly in magnitude and direction.

Causes:

  • Fluctuations in experimental conditions
  • Observer's limitations in reading
  • Unpredictable variations

Characteristics:

  • Cannot be completely eliminated
  • Can be reduced by taking multiple readings and averaging
  • Follow statistical laws

3. Gross Errors (Mistakes)

Definition: Errors due to carelessness or mistakes.

Examples: Reading wrong scale, recording wrong data, calculation mistakes

Remedy: Careful observation and cross-checking

Quantifying Errors

Absolute Error

Definition: Magnitude of difference between individual measurement and mean value.

If measurements are $a_1, a_2, ..., a_n$

Mean: $a_{mean} = \frac{a_1 + a_2 + ... + a_n}{n}$

Absolute errors: $\Delta a_1 = |a_1 - a_{mean}|$, $\Delta a_2 = |a_2 - a_{mean}|$, etc.

Mean absolute error: $\Delta a_{mean} = \frac{\Delta a_1 + \Delta a_2 + ... + \Delta a_n}{n}$

Result: $a = a_{mean} \pm \Delta a_{mean}$

Relative Error and Percentage Error

Relative Error: Ratio of mean absolute error to mean value

$$ \text{Relative Error} = \frac{\Delta a_{mean}}{a_{mean}} $$

Percentage Error: Relative error expressed as percentage

$$ \text{Percentage Error} = \frac{\Delta a_{mean}}{a_{mean}} \times 100\% $$
Worked Example 4 - Error Calculation

Q: A student measures length of a rod 5 times: 25.2 cm, 25.4 cm, 25.1 cm, 25.3 cm, 25.0 cm. Find (a) Mean length (b) Absolute errors (c) Mean absolute error (d) Percentage error.

Combination of Errors (Error Propagation)

Rule 1: Sum or Difference

If $Z = A + B$ or $Z = A - B$

Maximum absolute error in Z:

$$ \Delta Z = \Delta A + \Delta B $$

Note: Absolute errors ADD even for subtraction

Rule 2: Product or Quotient

If $Z = A \times B$ or $Z = \frac{A}{B}$

Maximum relative error in Z:

$$ \frac{\Delta Z}{Z} = \frac{\Delta A}{A} + \frac{\Delta B}{B} $$

Note: Relative errors ADD for both multiplication and division

Rule 3: Power

If $Z = A^n$ (where n can be positive, negative, or fraction)

Maximum relative error:

$$ \frac{\Delta Z}{Z} = n \frac{\Delta A}{A} $$

Or percentage error: $\%$ error in Z = $n \times$ (% error in A)

Worked Example 5 - Error Propagation

Significant Figures

Definition: All reliably known digits plus the first uncertain digit in a measurement.

Significant figures indicate the precision of measurement.

Rules for Counting Significant Figures:

  1. Non-zero digits: Always significant (123 has 3 s.f.)
  2. Zeros between non-zero digits: Always significant (1002 has 4 s.f.)
  3. Leading zeros: NOT significant (0.0052 has 2 s.f.)
  4. Trailing zeros in decimal: Significant (2.500 has 4 s.f.)
  5. Trailing zeros without decimal: Ambiguous (2500 has 2 or 4 s.f., use scientific notation to clarify)
  6. Exact numbers: Infinite s.f. (Ï€ = 3.14159...)

Examples:

  • 25.4 â†’ 3 s.f.
  • 0.00340 â†’ 3 s.f. (leading zeros don't count, trailing zero counts)
  • 6.023 Ã— 10Â²Â³ â†’ 4 s.f.
  • 300 â†’ 1 s.f. (ambiguous, write as 3.00 Ã— 10Â² for 3 s.f.)
  • 2.000 â†’ 4 s.f.

Rules for Calculations with Significant Figures:

Addition/Subtraction: Result should have decimal places equal to the least decimal places in any number.

Example: 12.11 + 18.0 + 1.012 = 31.122 â†’ Round to 31.1 (1 decimal place from 18.0)

Multiplication/Division: Result should have s.f. equal to the least s.f. in any number.

Example: 2.5 Ã— 3.42 Ã— 0.234 = 2.0007 â†’ Round to 2.0 (2 s.f. from 2.5)

01.4 Precision Measuring Instruments

Vernier Calipers

Vernier Calipers: Precision instrument for measuring internal/external dimensions and depths.

Components:

  • Main Scale: Fixed scale graduated in cm and mm
  • Vernier Scale: Moving scale that slides along main scale
  • External Jaws: For measuring outer dimensions
  • Internal Jaws: For measuring inner dimensions
  • Depth Probe: For measuring depths

Figure 1.1: Vernier Calipers - Scale Reading Illustration

Principle of Vernier:

10 Vernier Scale Divisions (VSD) = 9 Main Scale Divisions (MSD)

If 1 MSD = 1 mm, then:

10 VSD = 9 mm

1 VSD = 0.9 mm

Least Count (LC): Smallest measurement that can be measured accurately

LC = 1 MSD - 1 VSD = 1 mm - 0.9 mm = 0.1 mm = 0.01 cm

$$ LC = \frac{1 MSD}{Number \ of \ VSD} = \frac{1 mm}{10} = 0.1 mm$$

How to Read Vernier Calipers:

  1. Main Scale Reading (MSR): Note the position of zero of vernier scale on main scale. Read the mark just before zero of VS.
  2. Vernier Scale Reading (VSR): Find which vernier division coincides exactly with any main scale division. Multiply this number by LC.
  3. Total Reading: MSR + VSR
  4. Zero Error: Check if zero correction is needed

Zero Error in Vernier Calipers:

Positive Zero Error: When jaws are closed, if zero of VS is ahead (right) of zero of MS

Correction = Subtract zero error from reading

Negative Zero Error: When zero of VS is behind (left) of zero of MS

Correction = Add absolute value of zero error to reading

Worked Example 8 - Vernier Reading

Solution:

Screw Gauge (Micrometer)

Screw Gauge: Most precise instrument for measuring very small dimensions like wire diameter, sheet thickness.

Least Count: Typically 0.01 mm (10 times more precise than Vernier)

Components:

  • U-shaped Frame
  • Anvil: Fixed end
  • Spindle: Moving screw that rotates
  • Sleeve (Barrel): Fixed scale with main scale
  • Thimble: Rotating part with circular scale
  • Ratchet: Ensures constant pressure

Figure 1.2: Screw Gauge - Scale Reading Illustration

Screw Gauge Principle:

Pitch: Distance moved by spindle in one complete rotation

Typically, Pitch = 0.5 mm or 1 mm

Circular Scale Divisions: Usually 50 or 100 divisions

Least Count:

$$ LC = \frac{Pitch}{Number \ of \ Circular \ Scale \ Divisions} $$

If Pitch = 0.5 mm and CS divisions = 50:

$LC = \frac{0.5}{50} = 0.01$ mm

How to Read Screw Gauge:

  1. Linear Scale Reading (LSR) or PSR: Read the main scale mark visible just before the edge of thimble
  2. Circular Scale Reading (CSR): Note which division on circular scale coincides with the pitch line (reference line on sleeve)
  3. Total Reading: LSR + (CSR Ã— LC)
  4. Apply Zero Correction if needed

Zero Error in Screw Gauge:

Positive Zero Error: When spindle touches anvil, if zero of circular scale is below the pitch line

Reading shows n divisions below pitch line

Zero error = +n Ã— LC

Correction = Subtract from reading

Negative Zero Error: If zero is above pitch line

Zero error = -(100-n) Ã— LC (for 100-division scale)

Correction = Add absolute value to reading
