üîÑ 2. Rotational Symmetry

A figure has rotational symmetry if it looks the same after being rotated by some angle less than 360¬∞ about a fixed centre point.

Order of Rotational Symmetry: The number of times a figure looks the same during one complete rotation (360¬∞).

Angle of rotation = 360¬∞ √∑ Order of symmetry

‚Ä¢ Square: Order = 4, angle = 90¬∞
‚Ä¢ Equilateral triangle: Order = 3, angle = 120¬∞
‚Ä¢ Rectangle: Order = 2, angle = 180¬∞
‚Ä¢ Circle: Order = infinite (looks same at every angle)
‚Ä¢ Scalene triangle: Order = 1 (only at 360¬∞ ‚Üí no true rotational symmetry)
Centre of Rotation: The fixed point around which the figure is rotated.
Angle of Rotation: The angle through which the figure is rotated to look the same.
Note: Every figure has order 1 (at 360¬∞). "True" rotational symmetry requires order ‚â• 2.

  2. Regular polygon with n sides ‚Üí n lines of symmetry
  3. Rotational symmetry order = number of times figure looks same in one full (360¬∞) rotation

  5. Square: 4 lines, order 4 | Rectangle: 2 lines, order 2 | Rhombus: 2 lines, order 2
  6. Parallelogram: NO lines of symmetry, but order 2 rotational
