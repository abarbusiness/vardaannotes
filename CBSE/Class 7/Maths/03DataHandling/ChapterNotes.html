üìà 2. Mean, Median, Mode

Mean = Sum of all values √∑ Total number of values     ( xÃÑ = Œ£x/n )
Median = Middle value when data is arranged in ASCENDING order
Mode = Value that appears MOST frequently in the data
Example: Data = 4, 7, 3, 4, 8, 4, 7, 3
Arranged: 3, 3, 4, 4, 4, 7, 7, 8 (n=8)
Mean = (3+3+4+4+4+7+7+8) √∑ 8 = 40 √∑ 8 = 5
Median = average of 4th and 5th value = (4+4)/2 = 4
Mode = 4 (appears 3 times ‚Äî most frequent)
üß† When to use Mean, Median, Mode? ‚Ä¢ Mean: Best when data has no extreme values (no outliers). Use for exam averages.
‚Ä¢ Median: Best when there are extreme values (e.g., salary data with very rich people). Middle value is fair.
‚Ä¢ Mode: Best for finding the most popular item. E.g., most common shoe size, most popular subject.

üìâ 5. Probability ‚Äî Introduction

Probability of Event E = (Number of Favourable Outcomes) / (Total Possible Outcomes)
Key Points:
  • Probability ranges from 0 (impossible) to 1 (certain)
  • Random experiment: outcome cannot be predicted (e.g., tossing a coin, rolling a die)
  • Outcome: result of one trial
  • Event: a set of one or more outcomes
  • Probability of getting H on tossing a coin = 1/2 (1 favourable outcome out of 2 total)
  • Probability of getting 4 on rolling a die = 1/6 (1 favourable out of 6 total)
  • Probability of getting even number on die = 3/6 = 1/2 (2, 4, 6 are even)

üìù Quick Revision

  1. Mean = Sum √∑ Count. Best for balanced datasets without extreme values
  2. Median = Middle value (sorted). If even count, average the two middle values
  3. Mode = Most frequent value. A dataset can have no mode, one mode, or multiple modes
  4. Range = Maximum ‚àí Minimum
  5. Pie chart angle = (Value/Total) √ó 360¬∞
  6. Probability = Favourable Outcomes / Total Outcomes. Always between 0 and 1
