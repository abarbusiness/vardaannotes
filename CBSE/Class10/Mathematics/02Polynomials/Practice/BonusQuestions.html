Answer Key & Solutions: Polynomials

Class: 10 (CBSE) Subject: Mathematics Worksheet Solutions
SECTION A: OBJECTIVE TYPE (Answers)
1. Number of zeroes for 3 intersections:
Ans: (d) 3
Reason: The number of zeroes of a polynomial is equal to the number of points where its graph intersects the X-axis.
2. Value of α + β for x2 - 5x + 6:
Ans: (a) 5
Reason: Sum of zeroes α + β = -b/a = -(-5)/1 = 5.
3. Quadratic polynomial with zeroes -3 and 4:
Ans: (c) x2 - x - 12
Sum = -3 + 4 = 1. Product = -3 Ã— 4 = -12.
Polynomial = k[x2 - (Sum)x + Product] = x2 - 1x - 12.
4. Value of k if one zero is 2 for x2 + 3x + k:
Ans: (b) -10
Put x = 2 in polynomial: (2)2 + 3(2) + k = 0
4 + 6 + k = 0 ⇒ 10 + k = 0 ⇒ k = -10.
5. Shape of the graph of a quadratic polynomial:
Ans: (c) Parabola
6. Values of a and b if zeroes are 2 and -3:
Ans: (d) a = 0, b = -6
Sum of zeroes = 2 + (-3) = -1. From eqn: -(a+1) = -1 ⇒ a+1=1 ⇒ a=0.
Product of zeroes = 2(-3) = -6. From eqn: b = -6.
SECTION B: SHORT ANSWER (Solutions)
7. Find zeroes of 6x2 - 3 - 7x and verify.
Rewrite in standard form: 6x2 - 7x - 3.
Splitting middle term: 6x2 - 9x + 2x - 3 = 3x(2x - 3) + 1(2x - 3) = (3x + 1)(2x - 3).
Zeroes are x = -1/3 and x = 3/2.
Verification: Sum = -1/3 + 3/2 = 7/6. Formula (-b/a) = -(-7)/6 = 7/6. (Verified)
Product = (-1/3)(3/2) = -3/6 = -1/2. Formula (c/a) = -3/6 = -1/2. (Verified)
8. Quadratic polynomial with sum 1/4 and product -1.
Formula: k[x2 - (Sum)x + Product]
k[x2 - (1/4)x + (-1)]
To remove fraction, let k = 4: 4(x2 - x/4 - 1) = 4x2 - x - 4.
9. Zeroes are double of 2x2 - 5x - 3. Find p, q.
Zeroes of 2x2 - 5x - 3: Factorize 2x2 - 6x + x - 3 = 2x(x-3)+1(x-3) = (2x+1)(x-3). Zeroes are -1/2 and 3.
New Zeroes (Double): 2(-1/2) = -1 and 2(3) = 6.
New Polynomial: x2 - (-1+6)x + (-1)(6) = x2 - 5x - 6.
Comparing with x2 + px + q: p = -5, q = -6.
10. One zero of (k - 1)x2 + kx + 1 is -3. Find k.
Put x = -3 in the equation:
(k - 1)(-3)2 + k(-3) + 1 = 0
9(k - 1) - 3k + 1 = 0
9k - 9 - 3k + 1 = 0 ⇒ 6k - 8 = 0 ⇒ 6k = 8 ⇒ k = 4/3.
SECTION C: LONG ANSWER (Solutions)
11. f(x) = x2 - 5x + k and α - β = 1. Find k.
Here, a=1, b=-5, c=k.
Sum (α + β) = -(-5)/1 = 5. Product (αβ) = k.
Identity: (α - β)2 = (α + β)2 - 4αβ
(1)2 = (5)2 - 4(k)
1 = 25 - 4k ⇒ 4k = 24 ⇒ k = 6.
12. Form polynomial with zeroes 1/α, 1/β from 2x2 - 4x + 5.
Old zeroes: α + β = 4/2 = 2, αβ = 5/2.
New Sum S' = 1/α + 1/β = (β+α)/αβ = 2 / (5/2) = 4/5.
New Product P' = (1/α)(1/β) = 1/αβ = 1 / (5/2) = 2/5.
New Poly: k[x2 - (4/5)x + 2/5]. Take k=5: 5x2 - 4x + 2.
13. Squared difference of zeroes is 144 for x2 + px + 45.
Given (α - β)2 = 144.
Sum α + β = -p, Product αβ = 45.
Using (α - β)2 = (α + β)2 - 4αβ:
144 = (-p)2 - 4(45)
144 = p2 - 180 ⇒ p2 = 324 ⇒ p = ±18.
14. Find a, b for 2x2 - (a + 3)x + (2b - 1) if Sum=4, Prod=3.
Sum = -(-(a+3)) / 2 = (a+3)/2. Given Sum = 4.
(a+3)/2 = 4 ⇒ a+3 = 8 ⇒ a = 5.
Product = (2b-1)/2. Given Product = 3.
(2b-1)/2 = 3 ⇒ 2b-1 = 6 ⇒ 2b = 7 ⇒ b = 7/2 (or 3.5).
SECTION D: CASE STUDY (Solutions)
15. Case Study: P(x) = -x2 + 2x + 8
(i) Downwards. Reason: The coefficient of x2 is -1 (negative).
(ii) Find zeroes: -x2 + 2x + 8 = 0 ⇒ x2 - 2x - 8 = 0.
Factorize: (x - 4)(x + 2) = 0. Zeroes are 4 and -2.
(iii) Value at x = 0: P(0) = -(0)2 + 2(0) + 8 = 8.
(iv) Verify x = 4: P(4) = -(4)2 + 2(4) + 8 = -16 + 8 + 8 = 0. Yes, it is a zero.
