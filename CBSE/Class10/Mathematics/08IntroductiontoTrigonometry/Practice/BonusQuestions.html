Chapter Practice Sheet: Introduction to Trigonometry

Class: 10 (CBSE) Subject: Mathematics Max. Marks: 30
SECTION A: OBJECTIVE TYPE QUESTIONS (1 Mark Each)
1. If $\sin A = 3/4$, then $\cos A$ is:
  1. $4/3$
  2. $\sqrt{7}/4$
  3. $3/\sqrt{7}$
  4. $\sqrt{7}/3$
[1]
2. The value of $\tan 45^\circ + \cot 45^\circ$ is:
  1. 0
  2. 1
  3. 2
  4. $1/2$
[1]
3. If $\sin(A+B) = 1$ and $\cos(A-B) = \sqrt{3}/2$, $0^\circ < A+B \le 90^\circ, A>B$, then A is:
  1. $30^\circ$
  2. $45^\circ$
  3. $60^\circ$
  4. $90^\circ$
[1]
4. The value of $(\sin 30^\circ + \cos 30^\circ) - (\sin 60^\circ + \cos 60^\circ)$ is:
  1. -1
  2. 0
  3. 1
  4. 2
[1]
5. If $\tan A = \cot B$, then $A + B$ equals:
  1. $0^\circ$
  2. $90^\circ$
  3. $180^\circ$
  4. $45^\circ$
[1]
6. The value of $9 \sec^2 A - 9 \tan^2 A$ is:
  1. 1
  2. 9
  3. 8
  4. 0
[1]
SECTION B: SHORT ANSWER TYPE QUESTIONS (2 Marks Each)
7. Prove that: $(\sec A + \tan A)(1 - \sin A) = \cos A$.
[2]
8. If $\sin \theta = \cos \theta$, find the value of $2 \tan \theta + \cos^2 \theta$.
[2]
9. Evaluate: $2 \tan^2 45^\circ + \cos^2 30^\circ - \sin^2 60^\circ$.
[2]
10. Prove that $\sqrt{\frac{1+\sin A}{1-\sin A}} = \sec A + \tan A$.
[2]
SECTION C: LONG ANSWER TYPE QUESTIONS (3 Marks Each)
11. Prove that: $\frac{\tan \theta}{1-\cot \theta} + \frac{\cot \theta}{1-\tan \theta} = 1 + \sec \theta \csc \theta$.
[3]
12. If $\tan A + \sin A = m$ and $\tan A - \sin A = n$, show that $m^2 - n^2 = 4\sqrt{mn}$.
[3]
13. If $\sec \theta + \tan \theta = p$, prove that $\sin \theta = \frac{p^2-1}{p^2+1}$.
[3]
14. Evaluate without using trigonometric tables: $\frac{\sin^2 63^\circ + \sin^2 27^\circ}{\cos^2 17^\circ + \cos^2 73^\circ}$.
[3]
SECTION D: CASE STUDY (4 Marks)
15. Case Study: Right Triangle
In a right-angled triangle ABC, right-angled at B, if $\tan A = 1$ and AC = 10 cm.

(i) Find the measure of angle A. (1 Mark)
(ii) Find the length of AB. (2 Marks)
(iii) Evaluate $\sin A \cos C + \cos A \sin C$. (1 Mark)
[4]
