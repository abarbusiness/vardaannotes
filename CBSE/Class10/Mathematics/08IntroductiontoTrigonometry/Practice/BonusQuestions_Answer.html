Master Notes: Introduction to Trigonometry

Class: 10 (CBSE) Subject: Mathematics Worksheet Solutions
SECTION A: OBJECTIVE TYPE QUESTIONS
1. If $\sin A = 3/4$, then $\cos A$ is:
Ans: (b) $\sqrt{7}/4$
$\cos A = \sqrt{1 - \sin^2 A}$
$= \sqrt{1 - (3/4)^2} = \sqrt{1 - 9/16}$
$= \sqrt{7/16} = \sqrt{7}/4$.
2. The value of $\tan 45^\circ + \cot 45^\circ$ is:
Ans: (c) 2
$1 + 1 = 2$.
3. If $\sin(A+B) = 1$ and $\cos(A-B) = \sqrt{3}/2$... then A is:
Ans: (c) $60^\circ$
$A+B = 90^\circ$, $A-B = 30^\circ$.
Adding both: $2A = 120^\circ \Rightarrow A = 60^\circ$.
4. The value of $(\sin 30^\circ + \cos 30^\circ) - (\sin 60^\circ + \cos 60^\circ)$ is:
Ans: (b) 0
$(1/2 + \sqrt{3}/2) - (\sqrt{3}/2 + 1/2) = 0$.
5. If $\tan A = \cot B$, then $A + B$ equals:
Ans: (b) $90^\circ$
$\tan A = \tan(90^\circ - B) \Rightarrow A = 90^\circ - B \Rightarrow A+B = 90^\circ$.
6. The value of $9 \sec^2 A - 9 \tan^2 A$ is:
Ans: (b) 9
$9(\sec^2 A - \tan^2 A) = 9(1) = 9$.
SECTION B: SHORT ANSWER TYPE QUESTIONS
7. Prove that: $(\sec A + \tan A)(1 - \sin A) = \cos A$.
$(1/\cos A + \sin A/\cos A)(1 - \sin A)$
$= \frac{1+\sin A}{\cos A} (1-\sin A)$
$= \frac{1-\sin^2 A}{\cos A} = \frac{\cos^2 A}{\cos A} = \cos A$.
8. If $\sin \theta = \cos \theta$, find value of $2 \tan \theta + \cos^2 \theta$.
Ans: $2.5$ or $5/2$
$\tan \theta = 1 \Rightarrow \theta = 45^\circ$.
$2(1) + (1/\sqrt{2})^2 = 2 + 1/2 = 2.5$.
9. Evaluate: $2 \tan^2 45^\circ + \cos^2 30^\circ - \sin^2 60^\circ$.
Ans: 2
$2(1)^2 + (\frac{\sqrt{3}}{2})^2 - (\frac{\sqrt{3}}{2})^2 = 2$.
10. Prove that $\sqrt{\frac{1+\sin A}{1-\sin A}} = \sec A + \tan A$.
Multiply num and den by $\sqrt{1+\sin A}$.
$\sqrt{\frac{(1+\sin A)^2}{1-\sin^2 A}} = \frac{1+\sin A}{\cos A} = \sec A + \tan A$.
SECTION C: LONG ANSWER TYPE QUESTIONS
11. Prove that: $\frac{\tan \theta}{1-\cot \theta} + \frac{\cot \theta}{1-\tan \theta} = 1 + \sec \theta \csc \theta$.
Convert to sine/cos.
$\frac{\sin/\cos}{1-\cos/\sin} + \frac{\cos/\sin}{1-\sin/\cos} = \frac{\sin^2}{\cos(\sin-\cos)} + \frac{\cos^2}{\sin(\cos-\sin)}$
$= \frac{\sin^3 - \cos^3}{\sin \cos (\sin - \cos)}$
$= \frac{(\sin-\cos)(\sin^2+\cos^2+\sin\cos)}{\sin\cos(\sin-\cos)}$
$= \frac{1+\sin\cos}{\sin\cos} = \sec\theta\csc\theta + 1$.
12. If $\tan A + \sin A = m$ and $\tan A - \sin A = n$, show that $m^2 - n^2 = 4\sqrt{mn}$.
$m^2 - n^2 = (m+n)(m-n)$
$= (2\tan A)(2\sin A) = 4 \tan A \sin A$.
$4\sqrt{mn} = 4\sqrt{\tan^2 A - \sin^2 A}$
$= 4\sqrt{\frac{\sin^2 A}{\cos^2 A} - \sin^2 A}$
$= 4\sin A \sqrt{\sec^2 A - 1} = 4\sin A \tan A$.
Hence proved.
13. If $\sec \theta + \tan \theta = p$, prove that $\sin \theta = \frac{p^2-1}{p^2+1}$.
$\sec \theta - \tan \theta = 1/p$.
Adding: $2\sec\theta = p + 1/p = (p^2+1)/p$.
$\Rightarrow \cos\theta = 2p/(p^2+1)$.
Subtracting: $2\tan\theta = p - 1/p = (p^2-1)/p$.
$\sin\theta = \tan\theta/\sec\theta = \frac{(p^2-1)/2p}{(p^2+1)/2p} = \frac{p^2-1}{p^2+1}$.
14. Evaluate: $\frac{\sin^2 63^\circ + \sin^2 27^\circ}{\cos^2 17^\circ + \cos^2 73^\circ}$.
Ans: 1
$\sin 27^\circ = \cos 63^\circ$.
Num = $\sin^2 63 + \cos^2 63 = 1$.
$\cos 73^\circ = \sin 17^\circ$.
Den = $\cos^2 17 + \sin^2 17 = 1$.
Result = 1/1 = 1.
SECTION D: CASE STUDY
15. Case Study: Right Triangle
Detailed Solution:
(i) $\tan A = 1 \Rightarrow A = 45^\circ$.
(ii) $\sin 45^\circ = B/AC = BC/10 \Rightarrow 1/\sqrt{2} = BC/10 \Rightarrow BC = 5\sqrt{2}$.
Since $A=45, C=45, AB=BC=5\sqrt{2}$ cm.
(iii) $\sin A \cos C + \cos A \sin C = \sin(A+C) = \sin 90 = 1$.
