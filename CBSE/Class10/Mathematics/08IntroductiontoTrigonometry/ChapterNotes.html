3. Trigonometric Identities

Equations involving trigonometric ratios that are true for all values of the angles.

THE KING IDENTITY
$$ \sin^2 \theta + \cos^2 \theta = 1 $$

Derivatives: $\sin^2\theta = 1 - \cos^2\theta$ and $\cos^2\theta = 1 - \sin^2\theta$

Other Fundamental Identities

  • $1 + \tan^2 \theta = \sec^2 \theta$
  • $1 + \cot^2 \theta = \csc^2 \theta$

4. Important Exam Problems (PYQ Trends)

Type 1: Evaluating Expressions

Q: Evaluate $\frac{2 \tan 30}{1 - \tan^2 30}$.

Solution:

  • $\tan 30^\circ = 1/\sqrt{3}$.
  • Numerator: $2(1/\sqrt{3}) = 2/\sqrt{3}$.
  • Denominator: $1 - (1/\sqrt{3})^2 = 1 - 1/3 = 2/3$.
  • Result: $\frac{2/\sqrt{3}}{2/3} = \frac{2}{\sqrt{3}} \times \frac{3}{2} = \sqrt{3} = \tan 60^\circ$.

Type 2: Proving Identities

Q: Prove that $\sqrt{\frac{1+\sin A}{1-\sin A}} = \sec A + \tan A$.

Solution:

  1. Rationalize denominator inside root: Multiply num and den by $(1+\sin A)$.
  2. LHS $= \sqrt{\frac{(1+\sin A)^2}{1 - \sin^2 A}}$.
  3. Using $1-\sin^2 A = \cos^2 A$, LHS $= \sqrt{\frac{(1+\sin A)^2}{\cos^2 A}}$.
  4. Roots cancel squares: $\frac{1+\sin A}{\cos A}$.
  5. Split terms: $\frac{1}{\cos A} + \frac{\sin A}{\cos A} = \sec A + \tan A$. Hence Proved.
SYLLABUS NOTE Complementary Angles (e.g., $\sin(90-\theta) = \cos\theta$) are DELETED from the rationalized syllabus. Focus only on Ratios, Standard Angles, and Identities.
