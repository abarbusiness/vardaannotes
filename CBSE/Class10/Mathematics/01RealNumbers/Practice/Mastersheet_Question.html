Topic 5: Assertion-Reason (A-R) Questions
61.
Assertion (A): The HCF of two numbers is 5 and their product is 150. Then their LCM is 30.
Reason (R): For any two positive integers $a$ and $b$, $\text{HCF}(a,b) \times \text{LCM}(a,b) = a \times b$.
  1. Both A and R are true and R is the correct explanation of A.
  2. Both A and R are true but R is NOT the correct explanation of A.
  3. A is true but R is false.
  4. A is false but R is true.
62.
Assertion (A): $6^n$ ends with the digit 0, where $n$ is a natural number.
Reason (R): Any number ending with digit 0 must have both 2 and 5 as its prime factors.
  1. Both A and R are true and R is the correct explanation of A.
  2. Both A and R are true but R is NOT the correct explanation of A.
  3. A is true but R is false.
  4. A is false but R is true.
63.
Assertion (A): $\sqrt{2}$ is an irrational number.
Reason (R): If $p$ is a prime number and $p$ divides $a^2$, then $p$ divides $a$ (where $a$ is a positive integer).

  2. Both A and R are true but R is NOT the correct explanation of A.
  3. A is true but R is false.
  4. A is false but R is true.

  4. A is false but R is true.
