Chapter 5: Continuity and Differentiability

Dear Class 12 Student! Welcome to the gateway of advanced Calculus. In Class 11, we learned how to evaluate limits. Now, we use limits to check if a function is perfectly unbroken (Continuous) and if it has a smooth, predictable rate of change (Differentiable). This chapter requires absolute mastery of the Chain Rule and Logarithmic Differentiation, as they will haunt you in every subsequent chapter of Integration and Differential Equations. Let's trace the curve!

1. Continuity of a Function

Intuitive Idea: A function is considered continuous at a point if you can draw its graph around that point without ever lifting your pen from the paper. There are no breaks, jumps, or holes.

Continuity at a Point ($x=c$) A function $f(x)$ is strictly continuous at a point $x = c$ if and only if the following three conditions are met:
1. The function is defined at $x = c$ (i.e., $f(c)$ exists).
2. The limit of the function as $x \to c$ exists (i.e., $LHL = RHL$).
3. The value of the limit exactly equals the value of the function at that point. $$\lim_{x \to c^-} f(x) = \lim_{x \to c^+} f(x) = f(c)$$

Continuity in an Interval

  • Open Interval $(a, b)$: $f(x)$ is continuous at every single point lying between $a$ and $b$.
  • Closed Interval $[a, b]$: $f(x)$ must be continuous in $(a, b)$, and additionally:
    - It must be right-continuous at $a$: $\lim_{x \to a^+} f(x) = f(a)$.
    - It must be left-continuous at $b$: $\lim_{x \to b^-} f(x) = f(b)$.
Algorithm for Testing Continuity (RD Sharma Style) To check continuity at $x = c$:
Step 1: Find Left Hand Limit (LHL) by substituting $x = c - h$:
$\quad LHL = \lim_{h \to 0} f(c - h)$
Step 2: Find Right Hand Limit (RHL) by substituting $x = c + h$:
$\quad RHL = \lim_{h \to 0} f(c + h)$
Step 3: Find the exact value $f(c)$.
Step 4: Compare. If $LHL = RHL = f(c)$, it is continuous. Otherwise, it is discontinuous.
Practice Problem 1 Question: Check the continuity of the function $f(x)$ at $x = 2$, where:
$f(x) = \begin{cases} 2x + 3, & \text{if } x \le 2 \\ 2x - 3, & \text{if } x > 2 \end{cases}$
Solution:
We check at $x = 2$.
1. Value of function at $x=2$: Use the upper branch ($x \le 2$).
$f(2) = 2(2) + 3 = 4 + 3 = 7$.
2. LHL: $\lim_{x \to 2^-} f(x)$. Use the upper branch.
$LHL = \lim_{h \to 0} f(2 - h) = \lim_{h \to 0} [2(2 - h) + 3] = 2(2 - 0) + 3 = 7$.
3. RHL: $\lim_{x \to 2^+} f(x)$. Use the lower branch ($x > 2$).
$RHL = \lim_{h \to 0} f(2 + h) = \lim_{h \to 0} [2(2 + h) - 3] = 2(2 + 0) - 3 = 4 - 3 = 1$.
4. Since $LHL \neq RHL$ ($7 \neq 1$), the limit does not exist.
Therefore, $f(x)$ is discontinuous at $x = 2$.

2. Algebra of Continuous Functions

If $f(x)$ and $g(x)$ are two continuous functions at a real number $c$, then:

  • $f \pm g$ is continuous at $x = c$.
  • $f \cdot g$ is continuous at $x = c$.
  • $c \cdot f$ (scalar multiple) is continuous at $x = c$.
  • $f/g$ is continuous at $x = c$, provided $g(c) \neq 0$.
Continuity of Composite Functions If a function $g$ is continuous at a point $c$, and another function $f$ is continuous at the point $g(c)$, then their composite function $(f \circ g)(x) = f(g(x))$ is purely continuous at $c$.

Standard Continuous Functions (Always continuous in their domains): All Polynomials, Rational functions (where denom $\neq 0$), $\sin x$, $\cos x$, $e^x$, $\log x$, and $|x|$.
Practice Problem 2 Question: Prove that the function $F(x) = \sin(|x|)$ is a continuous function.
Solution:
Let us define two functions: $g(x) = |x|$ and $h(x) = \sin x$.
1. We know that the modulus function $g(x) = |x|$ is continuous for all real numbers.
2. We also know that the trigonometric sine function $h(x) = \sin x$ is continuous for all real numbers.
3. Let us find their composition $(h \circ g)(x)$:
$(h \circ g)(x) = h(g(x)) = h(|x|) = \sin(|x|) = F(x)$.
4. According to the theorem of composite functions, since $g(x)$ and $h(x)$ are continuous everywhere, their composition $F(x) = \sin(|x|)$ must also be continuous for all real numbers.

3. Differentiability (Derivability)

Differentiability geometrically means that the graph of the function has a unique, non-vertical tangent at that point. It implies "smoothness"—no sharp corners, kinks, or breaks.

Definition at a Point A function $f(x)$ is differentiable at $x = c$ if the derivative $f'(c)$ exists finitely:
$$f'(c) = \lim_{h \to 0} \frac{f(c+h) - f(c)}{h}$$ For this limit to exist, the Left Hand Derivative (LHD) must perfectly equal the Right Hand Derivative (RHD).

LHD: $\lim_{h \to 0^-} \frac{f(c-h) - f(c)}{-h}$
RHD: $\lim_{h \to 0^+} \frac{f(c+h) - f(c)}{h}$

Relation between Continuity and Differentiability

Theorem: If a function is differentiable at a point, it is necessarily continuous at that point. However, the converse is NOT true. Every continuous function is not necessarily differentiable.

The Classic Counter-Example: f(x) = |x|

The graph can be drawn without lifting the pen (Continuous at x=0). But it has a sharp corner at x=0, so no unique tangent can be drawn (Not Differentiable at x=0).

Practice Problem 3 Question: Prove mathematically that $f(x) = |x - 1|$ is continuous but not differentiable at $x = 1$.
Solution:
1. Continuity at $x=1$:
$f(1) = |1 - 1| = 0$.
$LHL = \lim_{h \to 0} f(1-h) = \lim_{h \to 0} |(1-h)-1| = \lim_{h \to 0} |-h| = 0$.
$RHL = \lim_{h \to 0} f(1+h) = \lim_{h \to 0} |(1+h)-1| = \lim_{h \to 0} |h| = 0$.
Since $LHL = RHL = f(1)$, $f(x)$ is continuous at $x=1$.

2. Differentiability at $x=1$:
$LHD = \lim_{h \to 0} \frac{f(1-h) - f(1)}{-h} = \lim_{h \to 0} \frac{|(1-h)-1| - 0}{-h} = \lim_{h \to 0} \frac{|-h|}{-h} = \lim_{h \to 0} \frac{h}{-h} = \mathbf{-1}$.
$RHD = \lim_{h \to 0} \frac{f(1+h) - f(1)}{h} = \lim_{h \to 0} \frac{|(1+h)-1| - 0}{h} = \lim_{h \to 0} \frac{|h|}{h} = \lim_{h \to 0} \frac{h}{h} = \mathbf{1}$.
Since $LHD \neq RHD$ ($-1 \neq 1$), $f(x)$ is not differentiable at $x=1$.

4. Rules of Differentiation (Chain Rule & Composite Functions)

The Chain Rule (Crucial Concept) The Chain Rule is used to differentiate composite functions ("functions inside functions").
If $y$ is a function of $u$ ($y = f(u)$) and $u$ is a function of $x$ ($u = g(x)$), then:
$$\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx}$$ You differentiate the "outside" function first, leaving the inside untouched, and then multiply by the derivative of the "inside" function.
Practice Problem 4 Question: Differentiate $y = \sin(\cos(x^2))$ with respect to $x$.
Solution:
This is a 3-layer composite function: $\sin(u)$, where $u = \cos(v)$, and $v = x^2$.
We apply the Chain Rule from outside to inside:
1. Derivative of outer $\sin(\dots)$ is $\cos(\dots)$.
$\frac{d}{dx}[\sin(\cos(x^2))] = \cos(\cos(x^2)) \cdot \frac{d}{dx}[\cos(x^2)]$
2. Derivative of middle $\cos(\dots)$ is $-\sin(\dots)$.
$= \cos(\cos(x^2)) \cdot [-\sin(x^2)] \cdot \frac{d}{dx}[x^2]$
3. Derivative of inner $x^2$ is $2x$.
$= \cos(\cos(x^2)) \cdot [-\sin(x^2)] \cdot (2x)$
Final Answer: $\frac{dy}{dx} = \mathbf{-2x \sin(x^2) \cos(\cos(x^2))}$.

5. Differentiation of Implicit Functions

Definition: When a relationship between $x$ and $y$ is expressed in a way that it is difficult or impossible to isolate $y$ on one side (e.g., $x^3 + y^3 = 3axy$), it is an implicit function.

Method: Differentiate both sides of the equation with respect to $x$. When differentiating any term containing $y$, you MUST apply the chain rule by multiplying with $\frac{dy}{dx}$. Finally, collect all $\frac{dy}{dx}$ terms on one side and solve for it.

Practice Problem 5 Question: Find $\frac{dy}{dx}$ if $x^3 + x^2y + xy^2 + y^3 = 81$.
Solution:
Differentiate both sides with respect to $x$. Use the Product Rule $(uv)' = u'v + uv'$ for the middle terms.
$\frac{d}{dx}(x^3) + \frac{d}{dx}(x^2y) + \frac{d}{dx}(xy^2) + \frac{d}{dx}(y^3) = \frac{d}{dx}(81)$
$3x^2 + \left[x^2 \frac{dy}{dx} + y(2x)\right] + \left[x(2y\frac{dy}{dx}) + y^2(1)\right] + 3y^2\frac{dy}{dx} = 0$
Rearrange and group all $\frac{dy}{dx}$ terms together:
$x^2\frac{dy}{dx} + 2xy\frac{dy}{dx} + 3y^2\frac{dy}{dx} = -3x^2 - 2xy - y^2$
Factor out $\frac{dy}{dx}$:
$\frac{dy}{dx} (x^2 + 2xy + 3y^2) = -(3x^2 + 2xy + y^2)$
$$\frac{dy}{dx} = \frac{-(3x^2 + 2xy + y^2)}{x^2 + 2xy + 3y^2}$$

6. Derivatives of Inverse Trigonometric Functions

Formula List (Must Memorize) - $\frac{d}{dx}(\sin^{-1} x) = \frac{1}{\sqrt{1-x^2}}$
- $\frac{d}{dx}(\cos^{-1} x) = -\frac{1}{\sqrt{1-x^2}}$
- $\frac{d}{dx}(\tan^{-1} x) = \frac{1}{1+x^2}$
- $\frac{d}{dx}(\cot^{-1} x) = -\frac{1}{1+x^2}$
- $\frac{d}{dx}(\sec^{-1} x) = \frac{1}{|x|\sqrt{x^2-1}}$
- $\frac{d}{dx}(\csc^{-1} x) = -\frac{1}{|x|\sqrt{x^2-1}}$

Simplification using Trigonometric Substitution: Differentiating a complex inverse trigonometric function directly using the chain rule becomes a nightmare. Instead, use the substitutions from Chapter 2 (ITF) to simplify the function before differentiating.

Practice Problem 6 Question: Differentiate $y = \tan^{-1}\left(\frac{\sin x}{1 + \cos x}\right)$ with respect to $x$.
Solution:
Do NOT use the chain rule directly. Simplify the inside expression first using half-angle formulas.
We know: $\sin x = 2\sin(x/2)\cos(x/2)$ and $1 + \cos x = 2\cos^2(x/2)$.
Substitute these in:
$y = \tan^{-1}\left( \frac{2\sin(x/2)\cos(x/2)}{2\cos^2(x/2)} \right)$
Cancel 2 and one $\cos(x/2)$:
$y = \tan^{-1}\left( \frac{\sin(x/2)}{\cos(x/2)} \right)$
$y = \tan^{-1}(\tan(x/2))$
Using self-adjusting property (assuming principal branch): $y = \frac{x}{2}$.
Now, differentiate with respect to $x$:
$\frac{dy}{dx} = \frac{d}{dx}(\frac{x}{2}) = \mathbf{\frac{1}{2}}$.

7. Exponential and Logarithmic Functions

Standard Derivatives:
- $\frac{d}{dx}(e^x) = e^x$
- $\frac{d}{dx}(a^x) = a^x \ln a$
- $\frac{d}{dx}(\ln x) = \frac{1}{x}$

Logarithmic Differentiation (Important Board Topic) This powerful method is used strictly in two scenarios:
1. Functions of the form Variable raised to a Variable (e.g., $y = x^{\sin x}$ or $y = (\sin x)^x$). *Warning: You CANNOT use $d/dx(x^n) = nx^{n-1}$ here!*
2. When a function is the product or quotient of many complex terms (e.g., $y = \sqrt{\frac{(x-1)(x-2)}{(x-3)(x-4)}}$).

Method:
Step 1: Take natural logarithm ($\ln$ or $\log_e$) on both sides.
Step 2: Use log properties ($\ln(a^b) = b\ln a, \ln(ab) = \ln a + \ln b, \ln(a/b) = \ln a - \ln b$) to completely expand and simplify.
Step 3: Differentiate implicitly with respect to $x$. (Remember $\frac{d}{dx}(\ln y) = \frac{1}{y}\frac{dy}{dx}$).
Step 4: Multiply the $y$ back to the right side and substitute the original function for $y$.
Practice Problem 7 Question: Differentiate $y = x^x$ with respect to $x$.
Solution:
This is of the form $(\text{variable})^{\text{variable}}$. We MUST use logarithmic differentiation.
1. Take natural log on both sides:
$\ln y = \ln(x^x)$
2. Apply property $\ln(a^b) = b\ln a$:
$\ln y = x \ln x$
3. Differentiate implicitly w.r.t $x$. Use Product Rule on RHS:
$\frac{1}{y} \frac{dy}{dx} = x \cdot \frac{d}{dx}(\ln x) + \ln x \cdot \frac{d}{dx}(x)$
$\frac{1}{y} \frac{dy}{dx} = x \cdot (\frac{1}{x}) + \ln x \cdot (1)$
$\frac{1}{y} \frac{dy}{dx} = 1 + \ln x$
4. Multiply $y$ to the RHS:
$\frac{dy}{dx} = y (1 + \ln x)$
5. Substitute back $y = x^x$:
$$\frac{dy}{dx} = x^x (1 + \ln x)$$

8. Derivatives of Functions in Parametric Forms

Definition: Sometimes, the relationship between $x$ and $y$ is expressed via a third, independent variable called a parameter (e.g., $t$ or $\theta$). For example, the parametric equations of a circle are $x = r \cos\theta, y = r \sin\theta$.

Formula: We differentiate both $x$ and $y$ separately with respect to the parameter, and then divide them.
$$\frac{dy}{dx} = \frac{dy/dt}{dx/dt}$$

Practice Problem 8 Question: Find $\frac{dy}{dx}$ if $x = a(\theta - \sin\theta)$ and $y = a(1 + \cos\theta)$.
Solution:
1. Differentiate $x$ w.r.t $\theta$:
$\frac{dx}{d\theta} = a \frac{d}{d\theta}(\theta - \sin\theta) = a(1 - \cos\theta)$.
2. Differentiate $y$ w.r.t $\theta$:
$\frac{dy}{d\theta} = a \frac{d}{d\theta}(1 + \cos\theta) = a(0 - \sin\theta) = -a\sin\theta$.
3. Apply parametric formula:
$\frac{dy}{dx} = \frac{dy/d\theta}{dx/d\theta} = \frac{-a\sin\theta}{a(1 - \cos\theta)} = \frac{-\sin\theta}{1 - \cos\theta}$.
4. Simplify using half-angle formulas ($\sin\theta = 2\sin(\theta/2)\cos(\theta/2)$ and $1-\cos\theta = 2\sin^2(\theta/2)$):
$\frac{dy}{dx} = \frac{-2\sin(\theta/2)\cos(\theta/2)}{2\sin^2(\theta/2)} = \frac{-\cos(\theta/2)}{\sin(\theta/2)} = \mathbf{-\cot(\theta/2)}$.

9. Second Order Derivative

Definition: Differentiating the first derivative ($dy/dx$) again with respect to $x$. It is denoted as $\frac{d^2y}{dx^2}$ or $y''$ or $f''(x)$ or $y_2$.

JEE Trap: Second Derivative in Parametric Form A very common mistake is assuming that $\frac{d^2y}{dx^2} = \frac{d^2y/dt^2}{d^2x/dt^2}$. THIS IS COMPLETELY WRONG!

The correct method: You must first find $\frac{dy}{dx}$ as a function of $t$. Then differentiate it with respect to $x$. Because it is a function of $t$, you MUST apply the chain rule and multiply by $\frac{dt}{dx}$ at the very end.
$$\frac{d^2y}{dx^2} = \frac{d}{dt}\left(\frac{dy}{dx}\right) \cdot \frac{dt}{dx}$$
Practice Problem 9 ("Prove That" Type) Question: If $y = A\sin x + B\cos x$, then prove that $\frac{d^2y}{dx^2} + y = 0$.
Solution:
1. Given: $y = A\sin x + B\cos x$.
2. Differentiate once w.r.t $x$ to get the first derivative:
$\frac{dy}{dx} = A\cos x + B(-\sin x) = A\cos x - B\sin x$.
3. Differentiate again w.r.t $x$ to get the second derivative:
$\frac{d^2y}{dx^2} = \frac{d}{dx}(A\cos x - B\sin x)$
$\frac{d^2y}{dx^2} = A(-\sin x) - B(\cos x)$
4. Factor out the negative sign:
$\frac{d^2y}{dx^2} = -(A\sin x + B\cos x)$
5. Notice that the term in the parenthesis is precisely our original function $y$. Substitute $y$ back in:
$\frac{d^2y}{dx^2} = -y$
6. Rearrange the equation:
$$\frac{d^2y}{dx^2} + y = 0$$ (Proved).

10. Mean Value Theorems (RD Sharma / RS Aggarwal Focus)

Note: These theorems link the average rate of change to the instantaneous rate of change. They are the theoretical underpinning of Calculus.

A. Rolle's Theorem

If a function $f(x)$ satisfies the following three conditions:
1. It is continuous in the closed interval $[a, b]$.
2. It is differentiable in the open interval $(a, b)$.
3. $f(a) = f(b)$ (The function returns to the same height).
Then, there exists at least one point $c \in (a, b)$ such that $f'(c) = 0$.

Geometric Interpretation: If a smooth, continuous curve starts and ends at the same y-value, it must have turned around at least once. At that turning point (peak or valley), the tangent line is perfectly horizontal (slope = 0).

B. Lagrange’s Mean Value Theorem (LMVT)

If a function $f(x)$ satisfies the following two conditions:
1. It is continuous in the closed interval $[a, b]$.
2. It is differentiable in the open interval $(a, b)$.
Then, there exists at least one point $c \in (a, b)$ such that: $$f'(c) = \frac{f(b) - f(a)}{b - a}$$

Geometric Interpretation: There is at least one point on the smooth curve where the tangent line is perfectly parallel to the secant line (chord) connecting the start point $(a, f(a))$ and end point $(b, f(b))$. Rolle's theorem is just a special case of LMVT where the chord is perfectly horizontal.

Practice Problem 10 Question: Verify Rolle's Theorem for the function $f(x) = x^2 - 4x + 3$ in the interval $[1, 3]$.
Solution:
Step 1: Check conditions.
1. $f(x) = x^2 - 4x + 3$ is a polynomial function. We know polynomials are continuous everywhere. So, it is continuous in $[1, 3]$.
2. Polynomials are differentiable everywhere. So, it is differentiable in $(1, 3)$.
3. Check $f(a) = f(b)$:
$f(1) = 1^2 - 4(1) + 3 = 1 - 4 + 3 = 0$.
$f(3) = 3^2 - 4(3) + 3 = 9 - 12 + 3 = 0$.
Since $f(1) = f(3)$, all three conditions of Rolle's theorem are strictly satisfied.

Step 2: Find $c$.
According to the theorem, there must be a $c \in (1, 3)$ where $f'(c) = 0$.
1. Differentiate $f(x)$: $f'(x) = 2x - 4$.
2. Set $f'(c) = 0 \implies 2c - 4 = 0$.
3. Solve for $c$: $2c = 4 \implies \mathbf{c = 2}$.
4. Check if $c$ lies in the interval: Yes, $2 \in (1, 3)$.
Hence, Rolle's Theorem is verified.
