Topic 2: Variable Separable Method
11.
Answer: $y \,dy = x \,dx \implies \int y \,dy = \int x \,dx \implies \frac{y^2}{2} = \frac{x^2}{2} + C \implies y^2 - x^2 = 2C$ (or $C'$).
12.
Answer: $\frac{dy}{dx} = e^x \cdot e^y \implies e^{-y} \,dy = e^x \,dx \implies \int e^{-y} \,dy = \int e^x \,dx \implies -e^{-y} = e^x + C \implies e^x + e^{-y} = K$.

14.

15.
